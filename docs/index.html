Why Blok

Built for developers,
designed for users

Everything you need to create powerful editing experiences in your applications.

Block Architecture

Content is structured as JSON data, not raw HTML. Parse, store, and render anywhere with ease.

Slash Commands

Built-in, customizable slash menu for quick formatting and inserting media.

Headless & Stylable

Blok gives you the logic; you bring the UI. Compatible with Tailwind, Styled Components, or raw CSS.

Drag & Drop

Native support for rearranging blocks with intuitive drag handles.

Extensible Plugin System

Create custom blocks for Kanbans, Embeds, Code Blocks, and more.

68 Languages

Lazy-loaded locale support with only English bundled by default (~3KB).

Quick Start

Up and running
in minutes

1

Install Blok

Add Blok to your project using your favorite package manager.

npm install @jackuait/blok
2

Import and configure

Import the editor and tools, then configure your block types.

import { Blok } from '@jackuait/blok';
import { Header, Paragraph, List, Bold, Italic, Link } from '@jackuait/blok/tools';

const editor = new Blok({
  holder: 'editor',
  tools: {
    paragraph: Paragraph,
    header: { class: Header, placeholder: 'Enter a heading' },
    list: List,
    bold: Bold,
    italic: Italic,
    link: Link,
  },
});
3

Save content

Extract clean JSON data ready to save anywhere.

const data = await editor.save();
// {
//   "version": "0.5.0",
//   "time": 1642697600000,
//   "blocks": [
//     {
//       "id": "abc123",
//       "type": "paragraph",
//       "data": { "text": "Hello World" }
//     }
//   ]
// }
